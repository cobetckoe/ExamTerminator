# Exam Terminator v1.0.0

高拍仪拍摄手机屏幕，PC端 OCR 识别题目，ADB 模拟点击自动作答。全程手机外部完成，软件层面无法检测。

## 功能

- 本地离线识别（RapidOCR）
- 高拍仪拍摄屏幕，OCR 自动识别
- 模糊匹配，容错率高
- 自动点击选项，全程无需手动
- Excel 题库管理
- 摄像头自动检测与切换
- 坐标校准，首次锁定后复用
- 参数自动保存
- 选项乱序兼容
- 判断题自动识别
- 可选多次识别取众数

## 硬件要求

- Windows 10/11
- 高拍仪或 USB 摄像头
- 小米手机（USB 调试）

## 题库格式

Excel 三列：题目 | 选项（`|` 分隔）| 答案（字母）

## 首版特性

- 完整的GUI界面
- 实时OCR识别
- 自动/手动答题模式
- 摄像头自动检测
- 参数自动保存
- 配置文件持久化
