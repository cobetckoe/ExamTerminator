## 功能

| 功能 | 说明 |
|------|------|
| **本地识别** | 基于 RapidOCR，完全离线，无需联网 |
| **自动识别** | 高拍仪拍摄手机屏幕，OCR 自动识别题目 |
| **智能匹配** | 模糊匹配算法，容错率高，支持前缀渐进匹配 |
| **自动答题** | 识别后自动点击选项，全程无需手动操作 |
| **题库管理** | Excel 格式，方便编辑和维护 |
| **摄像头选择** | 自动检测可用摄像头，支持在界面中切换 |
| **坐标校准** | 首次识别后锁定坐标系，后续题目复用 |
| **配置记忆** | 参数自动保存，下次启动无需重新设置 |
| **选项乱序兼容** | 按内容匹配选项，试卷选项顺序与题库不同也能答对 |
| **判断题支持** | 自动识别圆圈标识（○●）和文字选项，按内容匹配 |
| **多次识别** | 可选功能，截3次取众数，识别更准但速度较慢 |

## 硬件要求

| 设备 | 规格 |
|------|------|
| 系统 | Windows 10/11 |
| 摄像头 | 高拍仪（推荐）或 USB 摄像头 |
| 手机 | **小米手机**（开启 USB 调试） |

## 题库格式

准备 Excel 文件，格式如下：

| 题目 | 选项 | 答案 |
|------|------|------|
| 题目内容 | A. 选项1 \| B. 选项2 \| C. 选项3 \| D. 选项4 | A |

- 第 1 列：题目（支持模糊匹配）
- 第 2 列：选项（`|` 分隔）
- 第 3 列：答案（单选填字母 A/B/C/D，判断题填 A 或 B）

### 判断题格式

| 题目 | 选项 | 答案 |
|------|------|------|
| 判断题内容 | A. 正确 \| B. 错误 | A |

判断题选项内容为"正确/错误"、"对/错"、"√/×"时自动识别为判断题。

## 使用方法

### 第一次使用

1. 下载 `答题终结者.exe` 和 `题库模板.xlsx`（放在同一目录）
2. 按实际题目编辑题库
3. 连接高拍仪与小米手机
4. 手机开启 USB 调试模式
5. 双击运行 `答题终结者.exe`
6. 点击"浏览"加载题库
7. 调整参数（一般默认即可）
8. 点击"开始答题"

### 日常使用

1. 双击运行 `答题终结者.exe`
2. 加载题库（首次加载后会记住路径）
3. 点击"开始答题"

### 注意事项

- 首次识别后会锁定坐标系，后续题目请勿移动高拍仪或手机
- 自动模式仅支持小米手机，其他品牌请用手动模式
- 题前冷却默认 3 秒，符合考试间隔要求
- 识别失败时会自动切换到手动模式，可手动翻页后继续

## 从源码运行

```bash
git clone https://github.com/cobetckoe/ExamTerminator.git
cd ExamTerminator
pip install -r requirements.txt
python src/答题终结者.py
```

## 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 题目匹配 | 0.5 | 题目相似度阈值，越高越严格 |
| 选项匹配 | 0 | 选项相似度阈值，越高越严格 |
| 题前冷却 | 3.0 | 每道题开始前等待时间（秒），考试要求至少3秒 |
| 题后冷却 | 0 | 每道题结束后等待时间（秒） |
| 选项偏移 | 200 | 选项点击坐标Y轴偏移量 |
| 按钮偏移 | 200 | 下一题按钮坐标Y轴偏移量 |
| 多次识别 | 关 | 截3次取众数，识别更准但速度慢 |
